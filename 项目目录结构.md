# GitTimePrism 项目目录结构

```
GitTimePrism/
├── src/                              # 前端源代码目录（TypeScript + Vite）
│   ├── components/                   # UI 组件目录
│   │   ├── app.ts                    # 应用主框架组件（自定义标题栏、三栏布局、面板拖拽、壁纸功能、主题切换、撤销提交按钮）
│   │   ├── branch-list.ts            # 分支列表组件（显示本地/远程分支、切换分支）
│   │   ├── commit-detail.ts          # 提交详情组件（显示完整提交信息：哈希、作者、邮箱、日期、消息、文件变更列表）
│   │   ├── commit-graph.ts           # 提交节点图组件（可视化显示提交历史和分支关系）
│   │   ├── commit-input.ts           # 提交输入组件（输入提交消息、创建提交）
│   │   ├── diff-viewer.ts            # 差异对比视图组件（显示文件变更、行级 diff）
│   │   ├── file-list.ts              # 文件变更列表组件（显示暂存/未暂存文件、暂存/取消暂存操作）
│   │   ├── git-installer.ts          # Git 安装引导组件（检测 Git、显示安装引导）
│   │   ├── reset-dialog.ts           # 撤销提交对话框组件（选择撤销模式：soft/mixed/hard、二次确认）
│   │   └── terminal.ts               # 内嵌终端组件（xterm.js + PTY 交互）
│   ├── styles/                       # CSS 样式文件目录
│   │   ├── variables.css             # CSS 变量定义文件（Catppuccin Mocha/Latte 双主题 + 毛玻璃/透明效果参数）
│   │   ├── global.css                # CSS 重置和基础样式（滚动条、毛玻璃按钮、透明窗口、壁纸层）
│   │   └── components.css           # 组件样式（自定义标题栏、壁纸层、侧边栏、面板、状态栏等，全部支持毛玻璃透明效果）
│   ├── i18n/                         # 国际化语言包目录
│   │   ├── zh-CN.json                # 中文语言包
│   │   └── en-US.json                # 英文语言包
│   ├── services/                     # 服务模块目录
│   │   ├── i18n.ts                   # 国际化服务模块（多语言切换、翻译函数）
│   │   ├── wallpaper.ts              # 壁纸服务模块（壁纸选择/存储/加载/清除、Canvas采样+K-means主色调提取）
│   │   └── theme-engine.ts           # 动态变色引擎（根据壁纸主色调生成/应用/恢复CSS变量主题配色）
│   ├── utils/                        # 工具函数目录
│   │   └── dom.ts                    # DOM 操作工具函数（元素创建、事件委托、HTML设置）
│   └── main.ts                       # 应用入口文件（加载样式/i18n/App组件/事件监听）
├── src-tauri/                        # Rust 后端源代码目录（Tauri 2.x）
│   ├── src/                          # Rust 源代码目录
│   │   ├── commands/                 # Tauri IPC 命令模块（前端通过 invoke() 调用）
│   │   │   ├── mod.rs                # 命令模块入口（导出 system、terminal、remote、tag 等子模块）
│   │   │   ├── system.rs             # 系统命令（Git 检测、打开外部链接）
│   │   │   ├── terminal.rs           # 终端命令（PTY 创建/写入/调整/终止）
│   │   │   ├── remote.rs             # 远程操作命令（git pull 拉取更新、git push 推送提交）
│   │   │   └── tag.rs                # 标签管理命令（获取标签列表、创建/删除/切换标签）
│   │   ├── git/                      # Git 操作模块目录
│   │   │   ├── mod.rs                # Git 模块入口（导出所有 Git 子模块）
│   │   │   ├── commands.rs           # Git 命令执行辅助（run_git 函数、GitError 错误类型）
│   │   │   ├── repo.rs               # 仓库操作（克隆、初始化、状态检测）
│   │   │   ├── status.rs             # 工作区状态（git status 解析、文件变更列表）
│   │   │   ├── log.rs                # 提交历史（git log 解析、提交列表）
│   │   │   ├── graph.rs              # 提交节点图（git log --graph 解析、分支可视化）
│   │   │   ├── branch.rs             # 分支操作（分支列表、创建、删除）
│   │   │   ├── checkout.rs           # 切换操作（切换分支、检出提交）
│   │   │   ├── stage.rs              # 暂存操作（git add、git commit、git restore）
│   │   │   ├── diff.rs               # 差异对比（文件变更、行级 diff）
│   │   │   ├── pull.rs               # 拉取操作（git pull 从远程仓库拉取更新）
│   │   │   ├── push.rs               # 推送操作（git push 推送本地提交到远程仓库）
│   │   │   └── tag.rs                # 标签管理（获取标签列表、创建/删除/切换标签）
│   │   ├── utils/                    # 工具模块目录
│   │   │   ├── mod.rs                # 工具模块入口（导出 watcher、process 子模块）
│   │   │   ├── process.rs            # 进程管理辅助（静默执行命令、隐藏控制台窗口）
│   │   │   └── watcher.rs            # 文件监听（基础框架版本，后续接入仓库监听）
│   │   ├── lib.rs                    # 后端核心入口（注册命令/插件/状态/初始化逻辑）
│   │   └── main.rs                   # 桌面端入口（防止弹出控制台黑窗口）
│   ├── capabilities/                  # Tauri 2 权限配置目录
│   │   └── default.json              # 默认权限声明（窗口、事件、Shell、对话框、文件系统等）
│   ├── icons/                        # 应用图标目录
│   │   ├── icon.ico                  # Windows 图标
│   │   ├── icon.png                  # PNG 图标
│   │   └── ...                       # 其他尺寸图标
│   ├── Cargo.toml                    # Rust 依赖配置（tauri 2、portable-pty 0.8、serde 等）
│   ├── Cargo.lock                    # Rust 依赖锁定文件
│   ├── build.rs                      # Tauri 构建脚本
│   ├── tauri.conf.json               # Tauri 应用配置（窗口大小、CSP、打包等）
│   └── .gitignore                    # Rust 构建产物忽略配置
├── node_modules/                     # 前端依赖目录（npm install 生成，不提交 Git）
├── src-tauri/target/                 # Rust 编译输出目录（cargo build 生成，不提交 Git）
├── dist/                             # 前端构建输出目录（vite build 生成，不提交 Git）
├── docs/                             # 文档目录
│   ├── agents/                       # Agent 配置文档
│   ├── 项目目录结构.MD               # 项目目录结构说明
│   └── 项目规划文档.MD               # 项目完整规划文档
├── .trae/                            # Trae IDE 配置目录
│   ├── rules/                        # 项目规则目录
│   │   ├── git-commit-message.md     # Git 提交信息规范
│   │   └── 代码开发规范.md           # 代码开发规范
│   └── documents/                    # 计划文档目录
├── .agents/                          # Agent skills 目录
├── index.html                        # HTML 入口文件
├── package.json                       # NPM 依赖配置（@tauri-apps/api、@xterm/xterm 等）
├── package-lock.json                  # NPM 依赖锁定文件
├── tsconfig.json                      # TypeScript 编译器配置（ES2021 + 严格模式）
├── vite.config.js                     # Vite 构建工具配置（开发服务器 + 忽略 src-tauri 监听）
├── AGENTS.md                          # Agent 代理配置文件
├── .gitignore                         # Git 忽略配置（node_modules、target、dist 等）
└── 项目目录结构.md                    # 本文件 - 项目目录结构说明
```

## 目录说明

| 目录/文件 | 说明 |
|-----------|------|
| `src/` | 前端源代码，使用 TypeScript 编写，Vite 构建 |
| `src/components/` | UI 组件（App 主框架、Git 安装引导、终端面板） |
| `src/styles/` | CSS 样式（Catppuccin 双主题变量、全局重置、组件样式） |
| `src/i18n/` | 国际化语言包（中文 zh-CN、英文 en-US） |
| `src/services/` | 业务服务模块（国际化服务、壁纸服务、动态变色引擎） |
| `src/utils/` | 通用工具函数（DOM 操作辅助） |
| `src-tauri/` | Rust 后端源代码（Tauri 2.x 桌面应用框架） |
| `src-tauri/src/commands/` | Tauri IPC 命令（前端通过 `invoke()` 调用 Rust 函数，包括远程操作） |
| `src-tauri/src/git/` | Git 操作模块（仓库操作、状态查询、提交历史、节点图、分支管理、暂存、差异对比、拉取、推送） |
| `src-tauri/src/utils/` | Rust 工具模块（文件监听、进程管理） |
| `src-tauri/capabilities/` | Tauri 2 权限配置（窗口、事件、Shell、对话框、文件系统） |
| `src-tauri/Cargo.toml` | Rust 依赖配置 |
| `src-tauri/tauri.conf.json` | Tauri 应用配置（窗口 1280x800、无边框透明窗口、CSP允许data:图片） |
| `node_modules/` | 前端依赖（不提交 Git） |
| `dist/` | 前端构建输出（不提交 Git） |
| `src-tauri/target/` | Rust 编译输出（不提交 Git） |
| `docs/` | 项目文档（规划文档、目录结构等） |
| `package.json` | NPM 项目配置和依赖声明 |
| `tsconfig.json` | TypeScript 编译配置（ES2021 目标、严格模式） |
| `vite.config.js` | Vite 构建配置（开发服务器端口 5173、忽略 src-tauri 文件监听） |

## 技术栈

- **前端：** TypeScript + Vite 6 + xterm.js 6 + Catppuccin 主题 + 壁纸动态变色引擎
- **后端：** Rust + Tauri 2.x + portable-pty 0.8
- **特效：** 无边框透明窗口 + 毛玻璃半透明面板 + 壁纸驱动动态主题配色
