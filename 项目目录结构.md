# GitTimePrism 项目目录结构

```
GitTimePrism/
├── src/                          # 前端源代码目录（TypeScript）
│   ├── components/               # UI 组件目录
│   │   ├── app.ts                # 应用主框架组件（三栏布局、面板拖拽、主题切换）
│   │   ├── git-installer.ts      # Git 安装引导组件（检测 Git、显示安装引导）
│   │   └── terminal.ts          # 内嵌终端组件（xterm.js 集成，初始版本）
│   ├── styles/                   # CSS 样式文件目录
│   │   ├── variables.css         # CSS 变量定义文件（暗色/亮色双主题）
│   │   ├── global.css            # CSS 重置和基础样式（滚动条、按钮、全局容器）
│   │   └── components.css       # 组件样式（工具栏、侧边栏、面板、状态栏、终端等）
│   ├── i18n/                     # 国际化语言包目录
│   │   ├── zh-CN.json            # 中文语言包
│   │   └── en-US.json            # 英文语言包
│   ├── services/                 # 服务模块目录
│   │   └── i18n.ts              # 国际化服务模块（多语言切换、翻译函数、TypeScript 类型注解）
│   ├── utils/                    # 工具函数目录
│   │   └── dom.ts               # DOM 操作工具函数（元素创建、事件委托、HTML设置、TypeScript 类型注解）
│   └── main.ts                   # 应用入口文件（TypeScript 版本）
├── index.html                    # HTML 入口文件（script src 指向 main.ts）
├── tsconfig.json                 # TypeScript 编译器配置文件
├── vite.config.js                # Vite 构建工具配置文件
├── .trae/                        # Trae IDE 配置目录
│   └── rules/                    # 项目规则目录
│       ├── git-commit-message.md # Git 提交信息规范
│       └── 代码开发规范.md       # 代码开发规范
├── AGENTS.md                     # Agent 代理配置文件
└── 项目目录结构.md               # 本文件 - 项目目录结构说明
```

## 目录说明

| 目录/文件 | 说明 |
|-----------|------|
| `src/components/` | 存放 UI 组件，使用 TypeScript 编写，包含完整类型注解 |
| `src/styles/` | 存放所有 CSS 样式文件，包括主题变量、全局重置、组件样式 |
| `src/i18n/` | 存放多语言翻译文件，JSON 格式 |
| `src/services/` | 存放业务服务模块，如国际化服务等（TypeScript 版本） |
| `src/utils/` | 存放通用工具函数，如 DOM 操作辅助函数（TypeScript 版本） |
| `src/main.ts` | 应用入口文件，使用 TypeScript 编写 |
| `tsconfig.json` | TypeScript 编译器配置，ES2021 目标 + 严格模式 |
